Notes on Events:-
Events are mainly of two types:
Browser Events and Environment Events.

These events tell us things like:
Where the mouse was clicked
The mouse position (X and Y coordinates)
The window’s height and width (viewport)
The time when the event happened (timestamp)

*--event.type--*

It gives the browser which event happend like , click, keypressing(keydown), key-releaing (keyup)
Example- 
<p>Press any key or click the mouse to get the event type.</p>
<p id="log"></p>

function getEventType(event) {
  const log = document.getElementById("log");

  // Add the event type text on top of previous logs
  log.innerText = `${event.type}\n${log.innerText}`;
}

// Keyboard events
document.addEventListener("keydown", getEventType);
document.addEventListener("keypress", getEventType);
document.addEventListener("keyup", getEventType);

// Mouse events
document.addEventListener("mousedown", getEventType);
document.addEventListener("mouseup", getEventType);
document.addEventListener("click", getEventType);
